# Blog API (Go + PostgreSQL)

The backend for the blog app, built with Go (no frameworks) and PostgreSQL.

## Setup

1. Ensure PostgreSQL is running and `.env` is configured.
2. Run the server:

```bash
go run main.go db.go handlers.go models.go
```
